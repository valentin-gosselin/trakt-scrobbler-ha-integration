# Brand assets

Icon and logo for the Home Assistant integration.

- `icon.png` (256x256) and `icon@2x.png` (512x512): square app icon.
- `logo.png` (512x256) and `logo@2x.png` (1024x512): wide logo.

These are also submitted to [home-assistant/brands](https://github.com/home-assistant/brands)
under `custom_integrations/trakt_scrobbler/` so Home Assistant shows the icon on
the integration page (custom integrations load their icon from that repository,
not from the component itself).

The mark is an original design in Trakt's current visual style (purple gradient
with a checkmark); it is not the official Trakt logo.
